Amazon Web Services (AWS)

亞馬遜雲端運算(Amazon Web Service) 是一個高度可靠(Highly Reliable)，高擴充性(Highly Scalable)， 低成本(Low Cost)的基礎設施平台。 AWS在全球各地的客戶分布超過190個國家和地區，擁有數以千計的跨國公司，政府機構以及創業團隊。自從2006年推出以來，AWS利用建立在自己後端技術平台上的技術向廣大企業提供網絡服務。

技術創新一直是公司的核心文化，驅動著亞馬遜各部門的成長。經過十多年建設和運行運行巨大規模、高度可擴展的電子商務平台(Amazon.com) 的經驗，公司總結出了一套成功經營大規模技術基礎設施和數據中心的核心競爭力，併步入了一個更廣泛的新領域 -- 為開發人員和企業提供網路服務的平台。

如今，AWS平台包括30多個不同的服務，包括彈性計算(Elastic Compute Cloud)，存儲服務(Simple Storage Service)，數據庫服務(Relational Database Service)。 AWS的客戶可以選擇最適合其業務的發展平台或解決方案，進而把焦點放在企業創新，而不是花時間與金錢在維運硬體設備上。AWS的客戶包括高速成長的創業公司，例如Spotify，Dropbox和Pinterest，也包括成熟的大型公司，例如三星，殼牌，洲際酒店集團。當然也不乏台灣企業與新創團隊，例如TrendMico, D-Link, Pixnet, Cubie Messenger, iCook… 等。

AWS 目前在全球擁有多個數據中心（美國，巴西，歐洲，日本，新加坡和澳大利亞)，您可以選擇最適合貴公司業務市場的數據中心佈署服務。建議您可以透過 http://aws.amazon.com/free/ 註冊一個AWS帳號，並開始進行測試。

若您有任何AWS的相關問題，且希望跟AWS台灣團隊聯繫，請聯絡Locarno Pan locarnop@amazon.com 或是 James Tien jtien@amazon.com 。

KKBOX 音乐志》月刊，期望透过不同的媒体将音乐推向更广大的市场。而每年年初所举办的「KKBOX风云榜」颁奖典礼，因拥有每年超过50亿次点播数的公信力，更树立了KKBOX 在亚洲音乐业界的领导地位。2014 年KKBOX 推出「KKTIX售票亭」在线售票服务，以及「KKBOX Live音乐现场」系列演唱会，让会员除了在线聆听千万曲库，还能亲临现场享受近距离感动，提供全方位的娱乐体验和服务，满足乐迷的多样需求。

祐生研究基金會

祐生研究基金會自1978年12月30日設立登記以來，以對外一般碩士論文獎助，對內知識整理做為開端，經歷長年紮基的過程，在1987年起大力推動建築與環境永續發展相關的長期研究。在本會探究人類居住環境與永續發展的過程中發覺，近年來資訊科技與數位化的發展對於人類生活模式帶來了舉足輕重的影響，對於此領域之相關知識分享與整合，未來人才培育等等，是我們必需面對的重要課題之一。

因此，本會過去之碩士論文獎助主題除了與建築、環境、生態、健康相關議題以外，亦在2000至2008年間獎助「資訊社會與文化」類型之相關研究主題。而資訊科技與網路的發展為人類精神創發帶來新趨勢，所以在本會經過獎助方法與推廣模式的調整之後，目前首先針對國內獨立遊戲開發相關社群，推動小規模的知識與資訊分享聚會，將參訪國際遊戲開發論壇之研習心得與國內的獨立開發者交流，期能幫助國內相關社群之發展。就其他面向的知識整合來說，過去本會曾發展出知識庫、索引書編寫、養成各科門知識執守者等知識整合應用方式，作為進行主題研究及各項開展會議時消除知識盲點、精進研討結論的利器。目前定期舉辦聯誼會及讀書會，強化祐生成員的知識密度，以面對異質多變的未來。

《數位時代媒體群》創立於1999年7月，跨越20世紀，走過科技發展的關鍵年代，見證網路新經濟風起雲湧的演進歷程，我們持續探索、記錄、解讀全球正在發生，和即將發生的數位革命與變動。

《數位時代媒體群》為讀者尋找影響下一步的「新觀念」，描繪正在形成的「新環境」，記錄開創未來的「新典範」，深掘新商業模式的「新機會」及「新價值」，為讀者早一步解讀趨勢脈動，領先掌握下一步行動的競爭優勢。

凝聚全台最具影響力的科技菁英社群，集結台灣最具創造力的開發者及創業菁英，成為全台灣最重要的科技財經社群及知識傳播平台。

Linuxpilot

成立於1999年11月的Linux Pilot Ltd.，是一間以Linux及開放源碼軟體知識為本的公司，其產品及服務包括網站新聞資訊、刊物出版及培訓課程，務求以全方位方式滿足各界的學習需求。

而自2001年出版的Linuxpilot月刊，更是現時亞洲市場唯一的中文Linux期刊，至今已出版超過120期，廣為亞洲中文Linux用戶所愛戴。

癮科技

癮科技(COOL3C.COM)為台灣知名科技媒體網站，根據全球數據監測公司 comScore 測量，癮科技有每月近百萬名不重複使⽤用者(UV)。癮科技除了每天有12篇左右的國內外 最新3C產品、APP軟體消息,也與 網友分享許多科技⽣生活新知,另外更 透過許多深度分析、實測觀點的專欄，讓科技更貼近網友們的生活。

現在癮科技現在還致力於回答生活中「為什麼」的問題，例如「Youtube影片為什麼觀看次數都會留在301次好一陣子」這樣的小問題，解開了就會有一種爬上山頂喝上兩口涼水的唱快感呢！

PunNode

PunNode 是目前台灣最專注於科技創業的媒體，雖然我們在2014年2月10日才開始營運，但首月瀏覽量即逼近90萬，成為台灣科技創業媒體類的前兩名，顯示我們已經成功打入台灣科技創業圈，並有效引起話題與關注，而PunNode更是台灣唯一提供英文版的科技創業媒體。

目前PunNode的合作夥伴有TechNode動點科技、韓國主要科技網站Platum，團隊中擁有豐富的垂直領域內容經營經驗，我們期許能讓台灣的創業團隊與國際接軌，促成更多創業成功的案例。

Inside 硬塞的網路趨勢觀察

Inside是台灣科技媒體領導品牌，成立於2009年11月底，係由一群熱愛網路的人所成立的共筆部落格。我們專注於觀察創新、科技、新媒體與網路的趨勢發展，報導新創公司、創業家、網站、行動平台、社群媒體、行銷與數位內容。
